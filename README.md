# Data Visualization and Storytelling

## Overview

This repository documents my work and key learnings from a **Data Visualization and Storytelling** course in the Master’s program in **Business Analytics and Artificial Intelligence**.

The course focused on transforming complex datasets into **clear, compelling visual narratives** that support **data-driven decision making**. Through a series of hands-on projects, I explored how visualization techniques, interactive dashboards, and storytelling frameworks can be used to communicate insights effectively to technical and non-technical audiences.

The projects in this repository demonstrate practical applications of:

# Repository Projects

## 1. Data Storyboarding – Traffic Collision Narrative

This project explored how storytelling can enhance data communication.

I developed a narrative centered on a fictional character, **Alvin**, and integrated traffic collision statistics into the storyline. The storyboard illustrates how visualizations can be embedded within narrative sequences to guide readers through a structured data story.

**Key Concepts**

* Narrative structure in data communication
* Visual pacing and scene progression
* Integrating charts within storytelling contexts

### Design Process
- hand-drawn chart ideas
- Miro to logically organize data into buckets
- Figma for storboarding the final product
  
**Technologies Used**
- Figma
- Figma using the i Charts generate plugin (Interactive chart libraries)
- Miro (for data organization: Who, What, Where, When, Why)

## 2. Interactive Redesign of Florence Nightingale’s Visualization

This project modernizes **Florence Nightingale’s Coxcomb chart**, one of the earliest examples of data visualization used for policy advocacy.

The visualization was recreated and extended into an **interactive dashboard**, deployed through a Flask application.

### Design Process

- flourish to mockup Florence Nightingale's Rose Diagram
  
### Interactive Features

* Dropdown filters for **year and cause of death**
* Hover tooltips showing detailed data values
* Clickable chart segments generating a **corresponding line visualization**
* Dynamic scaling based on mortality counts

The project demonstrates how historical visualizations can be enhanced with modern interaction techniques.

**Technologies Used**

* Python
* Flask
* Data visualization dashboards
* Flourish
* Miro

## 4. Scrollytelling Story – The Rising Cost of Food

This project used **scrollytelling**, a narrative visualization technique where the story unfolds as users scroll through the page.

The story follows a student returning home for his weekly **strawberry pie**, illustrating how rising food prices affect everyday life.

The project integrates:

## 5. Evaluation of Statistics Canada Infographic

This project analyzed the effectiveness of Statistics Canada’s infographic:

**“Tariffs: No Impact Yet on Consumer Prices”**

The evaluation examined how the infographic communicates complex economic data related to Canadian tariffs on U.S. food products.

### Visualization Techniques Identified

* Side-by-side comparisons
* Icon-based representations of food ingredients
* Use of national flags to distinguish sources
* A **stacked hamburger visual metaphor** to represent ingredient pricing

### Key Findings

* The infographic uses **visual metaphor and iconography** to simplify complex economic data.
* The underlying data appears credible and is sourced, though details such as **sample size** are not explicitly discussed.
* The design effectively communicates price comparisons while maintaining strong visual engagement.
