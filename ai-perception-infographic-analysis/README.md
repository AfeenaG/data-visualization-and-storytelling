# AI Perception in Canada – Infographic Project Overview

This project explores how Canadians perceive and use Artificial Intelligence (AI) by analyzing survey data and transforming the findings into a clear and engaging infographic.

The objective of the assignment was to translate raw survey data into a visual narrative that communicates insights quickly and effectively to a general audience. The project emphasizes the use of visual hierarchy, layout design, and clear chart selection to present key findings.

Rather than focusing purely on data analysis, the primary goal was to design an infographic that communicates insights visually and intuitively.

# Dataset

The data used in this project comes from the following public dataset:

Borealis Data Repository

https://borealisdata.ca/dataset.xhtml?persistentId=doi:10.5683/SP3/WCUN0S

The dataset contains responses from Canadians regarding:

- Awareness of Artificial Intelligence

- Perceived benefits and risks of AI

- Personal use of AI technologies

- Attitudes toward AI adoption in society

These responses were analyzed to identify patterns in public perception and usage of AI technologies.

Design Process

Creating an effective infographic required a structured design workflow.

**1. Data Exploration**

The dataset was examined to identify key trends and insights that would be meaningful for a broad audience. This included identifying:

- major survey themes

- notable response patterns

- comparisons between attitudes and behaviors

**2. Initial Concept Sketches**

Hand-drawn sketches were used to explore possible infographic layouts and storytelling structures. Sketching allowed rapid experimentation with different layout ideas before committing to a digital design.

![Handsketch](Wireframes/01_Handsketch%20Of%20Infographic.jpg)

**3. Wireframing**

A digital wireframe was created to define the visual hierarchy and placement of charts, icons, and text. The wireframe helped organize the information into logical sections so the audience could quickly understand the key findings.


![Wireframe](Wireframes/02_WireFrame_How%20Canadians%20View%20and%20Use%20AI.png)

**4. Final Infographic**

The final infographic integrates the selected data insights into a visually structured design.

![Wireframe](Wireframes/03_How%20do%20Canadians%20View%20and%20Use%20AI.jpg)


The design prioritizes:

- clarity

- visual balance

- easy interpretation of survey results

Charts, icons, and layout elements work together to guide readers through the story of how Canadians view and interact with AI technologies.

# Key Learning Outcomes

This project reinforced several important principles of data visualization and infographic design.

# Visual Hierarchy is Critical

Information should be structured so viewers immediately recognize the most important insights.

Simplification Improves Communication

Infographics should focus on clear messages rather than excessive detail.

Design and Data Must Work Together

Effective communication requires balancing accurate data representation with strong visual design.

Planning Improves Results

Sketching and wireframing before final design helps ensure a coherent and engaging infographic.

# Why This Project Matters

Infographics are widely used in journalism, public policy, and business reporting because they allow audiences to quickly understand complex information.

This project demonstrates how survey data can be transformed into a clear, visually engaging format that supports public understanding of emerging technologies like AI.
